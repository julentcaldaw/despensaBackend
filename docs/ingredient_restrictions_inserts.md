# Inserts SQL para ingredient_restrictions

Este script asigna restricciones por ingrediente usando mapeo por nombre (sin IDs hardcodeados).

Supuestos usados:
- Alergenos y dietas etiquetados de forma conservadora.
- `Halal` y `Kosher` solo para ingredientes vegetales sin alcohol (evita falsos positivos).
- No se etiquetan automaticamente `Baja en sodio`, `Baja en azúcar`, `Baja en grasas saturadas`, `Keto`, `Paleo`, `Diabética` por requerir criterio nutricional más fino.
